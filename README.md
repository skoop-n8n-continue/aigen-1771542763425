# Weather Signage App

A minimalist, high-contrast weather application designed for digital signage.

## Features
- **Responsive 2:1 Layout**: Optimized for horizontal displays.
- **High Visibility**: Vibrant teal background with bold white typography.
- **Minimalist Design**: Flat aesthetic with a clean SVG sun animation.
- **Dynamic Content**: Automatically updates the date and time.
- **Cache-Busting**: Integrated meta tags and versioning to ensure real-time updates on signage hardware.

## Design Specs
- **Background**: #00b7af (Vibrant Teal)
- **Primary Font**: Inter
- **Aspect Ratio**: 2:1
